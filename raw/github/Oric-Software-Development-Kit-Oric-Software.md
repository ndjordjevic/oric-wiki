## README
Here is a repository of Oric software, games, tools and various routines made by various people. There are too many to give proper credit, so check either the folder names or the documentation in each project.

The repository organises content under two top-level trees:
- `public/oric/` — assets (charsets), demos (30years, AlchimieX, assembly_2002, buggy_boy, PushingTheEnvelope, _Zerofx), games (Awele, Dungeon Demo, Space 1999), hardware (cumulus), mym_player, routines (keyboard, rasterization with CircleBench/linebench/polybench, vortex), and tools.
- `users/` — per-author project folders: barnsey123, chema (1337, 1337intro, skooldaze, Elite Clone), coco, dbug (demos: 256bytes, doublebufferedsines, NyAtmos, stniccc2000, tunnel; games: 4kkong, cyclotron), jede, miri, mmu_man (demo/INVAL2K9, sedoric, ttf2oric), mooz, retromaster, torlus, twilighte (ImpossibleMission).

## Top-level structure
- `assets/` — shared graphic assets (character sets)
- `demos/` — Oric demo source code (Defence Force and others)
- `games/` — Oric game source code
- `hardware/` — hardware-related projects (e.g. Cumulus)
- `mym_player/` — MYM music format player code
- `routines/` — reusable Oric assembly routines (keyboard handling, rasterization benchmarks, vortex)
- `users/` — per-contributor project trees
- `readme.md` — repository overview

This repository is the GitHub home of the source code linked from the Defence Force games and demos pages — it replaced an earlier osdn.net SVN hosting (per the site's April 2025 changelog entry). Most projects build with the OSDK toolchain.
