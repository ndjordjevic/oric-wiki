## A Beginner's Guide To The Oric Atmos — https://blog.defence-force.org/index.php?page=articles&ref=ART56

Article published 13th October 2018. The author bought some Oric games and, from the same seller, found the Oric Special Autumn 1984 issue of Personal Software magazine. Inside was a Tansoft advertising insert and, inside that, a rare three-coloured document titled "A Beginner's Guide To The Oric Atmos".

Key content of the discovery:
- Personal Software Autumn 1984 is a 100-page magazine with Oric 1 and Atmos reviews, type-in programs and ads.
- Tansoft insert lists software (Defence Force £7.95, Ultima Zone £8.50, Oric Munch £7.95, Super-Advanced Breakout £4.50, Rat-Splat £7.95) and notes the Oric Atmos 48K at £170.00, Atmos 16K "T.B.A."
- The Beginner's Guide is a small document with front cover ("Help"), inside left page (connecting the computer to screen and tape), inside right page (confusing BASIC intro without code), and back cover (glossary including POKE definition).
- The Guide's style is reminiscent of The Hitchhiker's Guide to the Galaxy.

## Freeboot revisited — https://blog.defence-force.org/index.php?page=articles&ref=ART67

Article published 18th January 2020. Covers setting up a GoTek floppy emulator alongside a 3" drive on the Oric Microdisc.

Background: The author previously used an HxC (REV C) floppy emulator since 2013, but its form factor was impractical for the Microdisc. Decision: try the GoTek instead.

GoTek modifications required:
- Solder pins onto the internal connector (standard 2.54mm breakable row)
- Flash FlashFloppy firmware via USB-to-TTL interface (program jumper required)
- Install OLED screen (swap connector rotation 90°, unplug 7-digit display, plug OLED)
- Install piezo buzzer for disk-access audio feedback
- Install DPDT drive selector switch on the back plate to choose between 3" drive (A:) and GoTek (B:) or swap

Outcome: GoTek with FlashFloppy, OLED screen, buzzer, drive selector switch installed and working. FAT32 USB stick with HFE files (converted from DSK using HxC tools). Both drives accessible via jumper/switch; can boot from either.

## Encounter — One year later — https://blog.defence-force.org/index.php?page=articles&ref=ART88

Article published 15th December 2025. Full post-mortem of the Encounter game for the Oric, released 20th December 2024.

History:
- 1984: first attempt at the original Encounter (Severn Software, 1983) with CURSET graphics — ran out of memory.
- 2018: "Upgrade Time" YouTube dev-log series (17 videos, March–August 2018). Game rewritten from BASIC with graphical locations, custom font, leaderboard. Stalled in December 2018.
- 2019: burnout, project on hold.
- 2020: COVID period; first draft of C/assembler rewrite using FloppyBuilder codebase.
- 2022: reversed back to BASIC + assembler subroutines.
- March 2023: first Git commit of the final FloppyBuilder/C version on GitHub (github.com/Dhebug/Encounter).
- December 2024: released commercially on itch.io.

Technical details:
- Game engine written in 6502 assembly using OSDK and FloppyBuilder
- 42 unique graphical locations, monochrome 240×128 pixel art
- Custom adventure engine (not BASIC) loading images from floppy
- Localisations: English, French
- Development done in evenings/weekends alongside full-time job
- Git repository: https://github.com/Dhebug/Encounter
- Post-mortem includes detailed financial results, testing workflow, distribution channels (itch.io), and lessons learned about commercial retro game development

## Modernizing a Homemade PHP Blog With AI — https://blog.defence-force.org/index.php?page=articles&ref=ART91

Article published 23rd March 2026. Case study of using Claude Code to modernise the blog itself (blog.defence-force.org) — a flat-file PHP CMS running since 2009, ~1,000 lines of procedural PHP with no framework or database.

Changes made with AI assistance:
1. Set up local PHP development server (`winget install PHP.PHP.8.4`, `php -S localhost:8000`) — revealed deprecation warnings on PHP 8.5.
2. Fixed deprecated array_map/trim(null) pattern (7 instances) using array_pad() — backwards-compatible to PHP 4.
3. Added RSS feed: rss.php scanning 20 most recent articles, custom markup → plain text, first image extraction, <content:encoded> for full content.
4. Added OpenGraph and Twitter Card meta tags for social media previews (og:title, og:description, og:image, twitter:card). Fixed robots.txt Crawl-delay:25 blocking Twitter crawler.
5. Added article_url() helper appending slugified title to URLs (purely cosmetic, old URLs still work).
6. Big cleanup: identified duplicate articles (ART85–88 all identical), test/temp files, unreferenced images (~90 of 875), backup/import folder drafts.
