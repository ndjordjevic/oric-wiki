# Build journey

This folder is the long-form record of building a new Oric Atmos / Nova 64 — separate from the `/wiki` source pages, which are machine-generated by `/pin-llm-wiki`.

Each file is a dated, opinionated note about a decision, a build step, or a postmortem. They cite the wiki via `[[wikilinks]]` so context stays one click away.

## Conventions

- One file per decision or milestone, numbered `NN-short-name.md`.
- Front-matter at the top with `created`, `status`, and `related_wiki` (a list of `[[wikilink]]`s into `/wiki/sources/`).
- End each file with a **Decision log** section — append-only, one bullet per concrete action taken.
- Wikilinks go to `/wiki/sources/<slug>` pages. New external sources go through `inbox.md` + `/pin-llm-wiki run`, not directly into the wiki.
